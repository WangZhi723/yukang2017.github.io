Research

DetNAS: Neural Architecture Search on Object Detection [arXiv]
Yukang Chen*, Tong Yang*, Xiangyu Zhang, Gaofeng Meng, Chunhong Pan, Jian Sun.

- The first work to conduct NAS on object detecion.
- Analyse the gap between image classification and object detection through NAS.
- Meaningful architecture patterns.

EAT-NAS: Elastic Architecture Transfer for Accelerating Large-scale Neural Architecture Search [arXiv] [code]
Jiemin Fang*, Yukang Chen*, Xinbang Zhang, Qian Zhang, Chang Huang, Gaofeng Meng, Wenyu Liu, Xinggang Wang.

- Use architectures searched on small datasets as the initialization seeds for large-scale neural architecture search.
- Architecture-level transfer learning.
- 74.7% Top-1 accuracy on ImageNet.

Joint Neural Architecture Search and Quantization [arXiv] [code]
Yukang Chen, Gaofeng Meng, Qian Zhang, Xinbang Zhang, Liangchen Song, Shiming Xiang, Chunhong Pan.

- Joint neural architecture search and quantization.
- Provide existing networks with learning-based quantization policies.
- Adjustable trade-offs between accuracies and model size.
- 27.2% error rate with 4.9 MB on IMageNet.
- 2.97% error rate with 0.9 MB on CIFAR-10.

Reinforced Evolutionary Neural Architecture Search [CVPR 2019] [arXiv] [code]
Yukang Chen, Gaofeng Meng, Qian Zhang, Shiming Xiang, Chang Huang, Lisen Mu, Xinggang Wang.

- An integration of RL and EA in neural architecture search.
- 4 GPUs on 1.5 days for search.
- 75.7% Top1 accuracy on ImageNet.
- Superior to MobileNet, MobileNet-v2 and NASNet in semantic segmentation on the PASCAL VOC.
